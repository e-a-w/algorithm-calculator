O(n2) -- Quadratic

For input size and constant :

O( * 2) runs times.

Sample Code:

In this nested loop, the inner loop iterates through the entire array for every iteration of the outer loop. 

for (let i = 0; i < arr.length; i++) {
  for (let j = 0; j < arr.length; j++) {
      n++;
      console.log(`Loop #${n}`);
  }
}

O(n3) -- Cubic

For input size and constant :

O( * 3) runs times.

Sample Code:

Triply nested loop.

for (let i = 0; i < arr.length; i++) {
  for (let j = 0; j < arr.length; j++) {
    for (let k = 0; k < arr.length; k++)
      n++;
      console.log(`Loop #${n}`);
    }
  }
}
